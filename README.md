# crypto-asaqx645

A small reference repo for trading study and daily execution.

## What’s in here

- `crypto-terms.txt` - a plain-text glossary of crypto and trading terms, organized alphabetically from A-M and N-Z.
- `timezones.txt` - a quick reference for market sessions, opens/closes, and common timeframe abbreviations.
- `daily-routine-90-days.xlsx` - a 90-day routine workbook with weekly tabs, dated weekday columns, and checkbox-style habit tracking.
- `skills-folder/notes.txt` - informal notes about money goals, building skills, and time management.

## Daily Routine Workbook

The Excel workbook is organized into 14 weekly sheets:

- `Week 1` starts on Thursday, April 9, 2026.
- `Week 14` ends on Tuesday, July 7, 2026.
- The workbook uses weekday-only columns.
- Habit cells are formatted as checkboxes so the same checklist can be reused across all tabs.

The routine focuses on:

- Morning Plan
- Stretching and yoga
- Running
- Work hours
- TRW study
- Calisthenics
- End of Day Review

## How To Use

1. Open `daily-routine-90-days.xlsx` in Excel.
2. Use each week tab to track your daily habit completion.
3. Keep `crypto-terms.txt` and `timezones.txt` open as quick study references while you work.

## Notes

- This repo does not require any build steps.
- The workbook file is the main active artifact in the project.
- `skills-folder/notes.txt` is a rough scratchpad rather than a finished document.
